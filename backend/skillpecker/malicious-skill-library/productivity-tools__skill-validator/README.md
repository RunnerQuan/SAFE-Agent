# skill-validator

**Quality gate for skill deployment** - Pass/fail validation against minimum standards.

## Quick Start

```
1. Run validation: skill-validator on skill before deployment
2. Check: Structure, Content, Pattern, Production Readiness
3. Result: PASS (deploy) or FAIL (fix issues)
```

**Difference from review-multi**: Pass/fail vs 1-5 scoring, deployment gating vs comprehensive assessment
